So'al

  1. Kenapa kita harus menyimpan file test dengan ekstensi .php?
  2. Apakah jika file test dirubah ekstensinya menjadi .html program akan tetap menghasilkan output yang diinginkan ?
  3. Apa fungsi folder htdocs apakah semua file php harus disimpan didalam folder tersebut ?
  4. Apa yang terjadi apabila menjalankan file php tidak melalui url localhost ?
  5. Apa yang menjadi pembeda antara script HTML dan PHP didalam konsep pemrograman web ?

Jawaban

  1. Biasanya, file dengan ekstensi .php digunakan untuk menyimpan kode PHP. PHP adalah bahasa pemrograman server-side yang digunakan untuk membuat halaman web dinamis. Dengan menyimpan file dengan ekstensi .php, server web akan tahu bahwa file tersebut berisi kode PHP yang harus dieksekusi sebelum dikirimkan ke browser

  2. Jika sebuah file dengan ekstensi .php diubah menjadi .html, program PHP di dalamnya mungkin tidak akan dieksekusi oleh server web secara otomatis. Sebagai gantinya, isi file tersebut akan dianggap sebagai kode HTML biasa oleh server, dan outputnya mungkin tidak sesuai dengan yang diharapkan jika ada kode PHP di dalamnya

  3. File PHP seharusnya disimpan di dalam folder "htdocs" jika ingin file PHP dapat diakses melalui web server. Ketik ingin menjalankan server web lokal seperti Apache, folder "htdocs" adalah tempat di mana Anda meletakkan file-file yang ingin akan di akses melalui browser. Tidak semua file PHP harus disimpan di dalam folder htdocs, tetapi file-file yang ingin diakses melalui web (seperti halaman web) harus ditempatkan di dalamnya agar dapat diakses oleh pengguna melalui URL.

  4. Jika menjalankan file PHP tanpa melalui URL localhost atau server web, kode PHP tidak akan dieksekusi. Kode PHP hanya dieksekusi oleh server web yang memiliki interpreter PHP. Jadi, jika menjalankan file PHP secara langsung dari file sistem lokal (misalnya, dengan membuka file tersebut di browser langsung dari sistem file), maka kode PHP tidak akan diinterpretasikan dan hanya kode HTML biasa yang akan ditampilkan.

  5. HTML digunakan untuk membuat struktur dan konten halaman web. HTML adalah bahasa markup yang digunakan untuk mengatur elemen seperti teks, gambar, link, tabel, formulir, dan lainnya. HTML berfungsi sebagai kerangka dasar untuk menampilkan informasi di browser. PHP adalah bahasa pemrograman server-side yang digunakan untuk menghasilkan konten dinamis dalam halaman web. PHP memungkinkan Anda untuk melakukan pemrosesan data, menghubungkan ke database, mengelola formulir, dan menghasilkan output yang berbeda berdasarkan kondisi atau input tertentu. PHP berjalan di sisi server dan menghasilkan HTML yang kemudian dikirim ke browser untuk ditampilkan. Perbedaan utama antara script HTML dan PHP adalah bahwa HTML adalah bahasa markup yang digunakan untuk membuat struktur halaman web, sedangkan PHP adalah bahasa pemrograman server-side yang digunakan untuk membuat halaman web dinamis. HTML hanya dapat membuat halaman web dengan konten statis, sementara PHP memungkinkan Anda untuk membuat halaman web yang dapat berinteraksi dengan basis data, mengolah formulir, menghasilkan konten dinamis berdasarkan parameter tertentu, dan melakukan banyak fungsi lainnya secara dinamis.
